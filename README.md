# AcSapfir

Плагин AutoCAD 2020 для экспорта геометрии в САПФИР.

## Требования

- AutoCAD 2020
- САПФИР 2018+
- .NET Framework 4.8

## Установка

1. Собрать `AcSapfir.dll` (Debug или Release)
2. В AutoCAD выполнить `NETLOAD` и выбрать `bin\Debug\AcSapfir.dll`
3. Использовать команды из списка ниже

## Команды

| Команда | Описание |
|---|---|
| `Sapfir_AXES` | Координационные оси из отрезков (нумерация по порядку) |
| `Sapfir_AXES_WIZ` | Мастер осей: авто-группировка параллельных линий, 4 правила имён |
| `Sapfir_Point_LOADS` | Точечные нагрузки из точек |
| `Sapfir_Line_LOADS` | Равномерные линейные нагрузки из отрезков |
| `Sapfir_Line_LOADS2` | Трапециевидные линейные нагрузки из отрезков |
| `Sapfir_SLABS` | Плиты перекрытия из полилиний |
| `Sapfir_SLAB_Holes` | Отверстия в плитах из полилиний |
| `Sapfir_Found_SLABS` | Фундаментные плиты из полилиний |
| `Sapfir_WALLS` | Стены из отрезков |
| `Sapfir_WALL_EDIT` | Редактирование существующих стен (обновление геометрии по XData) |
| `Sapfir_DOORS` | Дверные проёмы в стенах из отрезков |
| `Sapfir_WINDOWS` | Оконные проёмы в стенах из отрезков |
| `Sapfir_COLUMNS` | Колонны из полилиний (авто/ручной угол поворота) |
| `Sapfir_BEAMS` | Балки из отрезков с сечением h×b |
| `Sapfir_LINES` | Линии построения из отрезков AutoCAD |
| `Sapfir_STOREYS` | Этажи из MText (имя + Y-отметка) |
| `Sapfir_STOREYS_WIZ` | Мастер генерации этажей (модальное окно, шаблоны) |

## Возможности

- **Поддержка вкладок САПФИР** — команды работают с активной вкладкой через `GetActiveSapfirView()`
- **XData-трекинг** — созданные ID объектов САПФИР сохраняются обратно в примитивы AutoCAD для последующего редактирования
- **Выбор материала** — модальный диалог выбора материала из проекта САПФИР перед созданием элементов
- **Логирование в консоль** — все команды выводят ID созданных объектов в консоль AutoCAD (F2)

## Масштабирование координат

Геометрия масштабируется из миллиметров (AutoCAD) в метры (САПФИР) с коэффициентом 0.001.

Отметка этажа — абсолютная в метрах: `newStorey.Parameter["M_LEVEL"] = отметка_м`.

## Сборка

```bash
msbuild AcSapfir.csproj /t:Build /p:Configuration=Debug
```

Результат: `bin\Debug\AcSapfir.dll`
